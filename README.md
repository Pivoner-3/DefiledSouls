# DefiledSouls

Ты - один из легендарной четвёрки приключенцев, которые погибли после встречи с демоном древнего зла. После смерти ты очутился в чистилище, и после разговора с божеством ты понимаешь, что не собираешься умирать навсегда, и собираешься вернуться к жизни. 
Во время пребывания в чистилище, игрок будет привыкать к механике игры, учиться взаимодействовать с предметами и так далее, а после того как найдет портал в настоящий мир, можно будет выбрать один из классов (один из тех четырех). После этого начнется основная игра.
